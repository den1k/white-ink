# White Ink

### DEV

to run, cd into dir and:
```
lein figwheel
```

Once it compiled, go to localhost:3450.


### App Usage

White Ink is minimal in it's ui and shortcuts.
To move outside of the editor one menu interface currently only accessible
by shortcut is provided.
Shortcut to access the menu: `option + /`

There you can click on the shortcuts menu to see the other app shortcuts.
